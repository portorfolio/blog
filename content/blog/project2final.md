## My contribution
I was responsible for handling the backend (server.js) and altered a bit of the front end to make sure the backend was working properly. I adjusted the css to make sure the nav bar worked in our favor and for content to display when the user clicks into a specific album. The form.njk was tweaked to allow for information to be uploaded.
